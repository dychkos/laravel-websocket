
# Laravel Websocket

Base example using Websocket on Laravel.


## Installation

Clone repository using git clone

```bash
  git clone {repo_url}
```

Install all required dependencies.

```bash
  composer install
  npm install
```

Run migrations

```bash
  php artisan migrate
```

Start development server

```bash
  npm run dev
  php artisan server
```

Start websocket server

```bash
  php artisan websockets:serve
```
